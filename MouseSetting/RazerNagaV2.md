# マウス設定一覧

## 基本ボタン

| 操作               | 割り当て             |
|--------------------|----------------------|
| 左クリック         | 左クリック           |
| 右クリック         | 右クリック           |
| 上スクロール       | 上スクロール         |
| 下スクロール       | 下スクロール         |
| スクロールクリック | スクロールクリック   |
| スクロール右       | Num Enter            |
| スクロール左       | Delete               |

## 上部追加ボタン

| ボタン      | 割り当て |
|-------------|----------|
| 上部ボタン1 | Ctrl + C |
| 上部ボタン2 | Ctrl + V |

## サイドボタン（1〜12）

### 左側（1〜6）

| ボタン番号 | 割り当て    |
|------------|-------------|
| 1          | Space       |
| 2          | Alt + Right |
| 3          | 半角/全角   |
| 4          | Alt + D     |
| 5          | Alt + Left  |
| 6          | Ctrl        |

### 右側（7〜12）

| ボタン番号 | 割り当て |
|------------|----------|
| 7          | Ctrl + W |
| 8          | Shift    |
| 9          | Alt      |
| 10         | V        |
| 11         | Windows  |
| 12         | Tab      |
